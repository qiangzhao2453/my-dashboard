🛠️ 技术栈

前端：HTML5 + CSS3 + JavaScript（ES6+）
图表库：ECharts 5.x
数据源：CSV格式原始数据
开发方式：Claude Code辅助编程 + 自主调试
部署：GitHub Pages

🚀 本地使用

Clone或下载本仓库
直接双击 index.html 即可在浏览器中打开
使用年份切换按钮查看2024/2025年度数据
无需安装任何依赖，无需联网
